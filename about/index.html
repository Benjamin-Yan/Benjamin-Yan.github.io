# 📜 緣起

高中時開始對程式設計產生興趣，到了大學才真正進入日文音樂，Vtuber 和動漫的世界。

想當初大一之時，心血來潮就在暑假的時候在 Blogger 上開始用 HTML , CSS 及 JS 這些前端語言寫一些網頁做練習，在這期間也斷斷續續的寫了一些 Python 的小工具，後來大二課業壓力加重後就漸漸沒有更新了。

某天偶然滑到用此主題架設的網站，覺得這個風格我很喜歡，於是就開始學習用 Hexo 框架架設我的網站，一路持續至今。(除了大三做專題時比較沒在更新)，也是在這時我開始寫一些網頁小工具，幫助我的一些日常任務。

這個網站的誕生，源於自己對這些興趣的熱愛，希望能夠記錄這段旅程，也能與同樣熱愛這些事物的朋友們交流！
