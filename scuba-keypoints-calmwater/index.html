# 手臂搜尋

  • 吸一口氣讓肺部擁有足夠空氣。
  • 調節器離開嘴巴時，應保持吐氣發出阿聲。
  • 咬嘴朝下，將二級頭輕輕放下，不要用甩。
  • 用右手臂沿著大腿邊緣往外橫掃。
  • 用左手找回二級頭，排水後小心呼吸道控制。

# 順管摸索

  • 吸一口氣讓肺部擁有足夠空氣。
  • 調節器離開嘴巴時，應保持吐氣發出阿聲。
  • 咬嘴朝下，將二級頭輕輕放下，不要用甩。
  • 用左手將氣瓶底座往上托高。
  • 右手往後找到調節器一級頭，再順著管子往下找到二級頭，排水後小心呼吸道控制。

# 面鏡部份進水的排水

  • 吸一口氣讓肺部擁有足夠空氣。
  • 將面鏡輕輕拉開讓水進滿一半。
  • 鼻子輕輕吐氣後，把頭開始往上抬。
  • 眼睛應睜開，戴隱形眼鏡者應將眼睛閉上。
  • 有排水閥設計者應將排水閥置於最低處。

# 備用空氣源呼吸 (原地)

  • 受氣者（沒有空氣請給空氣的手勢）。
  • 找到同伴的備用空氣源後將自己二級頭拿開。
  • 咬嘴不要咬反。
  • 同伴應保持身體的接觸（右手抓著右手）。
  • 左手將 BCD 充氣閥舉高舉直準備洩氣（上昇時）。
  • 連續呼吸 30 秒以上，角色互換。

# 水中面鏡回復

  • 吸一口氣讓肺部擁有足夠空氣。
  • 將面鏡輕輕拉開讓水進滿後再將面鏡拿下。
  • 面鏡抓在手上，調節器不要吐掉。
  • 戴回時前額的頭髮不要糾纏，面鏡不要戴反。
  • 排水完後檢查面鏡帶有無糾纏。

# 空氣耗盡的體驗

  • 教練給學員一個 OK 手勢，將氣瓶閥慢慢關上。
  • 學員一邊呼吸一邊注視住壓力錶的指針變化。
  • 直到呼吸不到空氣時，學員再給教練空氣耗盡的手勢，教練再把氣瓶閥慢慢打開。

# 控制式緊急游泳上昇（水平方向）

  • BCD 建立正浮力。
  • 右手往前舉，左手將 BCD 充氣閥舉高。
  • 吸一口氣模擬沒有空氣（調節器不要吐掉）。
  • 保持一直吐氣並發出阿聲。
  • 正常游動速度，不能偷吸氣。
  • 至少游動 9 公尺。

# 中性浮力檢視

# 蛙鞋旋軸式（使用低壓充氣閥）

  • 先將 BCD 內部空氣洩光。
  • 身體平趴在水底利用蛙鞋鞋尖做轉軸。
  • 左手握著 BCD 充氣閥做點放式充氣。
  • 充完氣後開始呼吸（重複 3-4 次循環後）。
  • 直到吸氣時身體會開始上揚，吐氣時身體會慢慢下沉（身體不能碰觸地面）。
  • 不可憋氣，不可利用雙腳踹動來幫助平衡。

# 蛙鞋旋軸式（使用口吹）

  • 將 BCD 內部空氣洩光。
  • 身體平趴在水底利用蛙鞋鞋尖做轉軸。
  • 吸一口氣後將二級頭拿離開嘴巴，左手將 BCD 充氣閥送入嘴巴，按住洩氣鈕並向內吹氣。
  • 吹完氣後將洩氣按鈕放開，並咬回調節器重新開始呼吸（重複 2-3 次循環後）。
  • 直到吸氣時身體會開始上揚，吐氣時身體會慢慢下沉（身體不能碰觸地面）。
  • 不可憋氣，不可利用雙腳踹動來維持平衡。

# 同伴共生呼吸 (原地)

  • 受氣者：沒有空氣請給我空氣的手勢。
  • 供氣者應處在右手邊的位置。
  • 受氣者抓住供氣者的右手臂（不要擋住出氣按鈕）。
  • 維持每次呼吸二口氣（咬嘴不要咬反）。
  • 調節器離開嘴巴時應保持吐氣。
  • 同伴儘量保持身體的接觸。
  • 角色互換。

# 大量漏氣調節器呼吸

  • 吸一口氣將調節器拿離開嘴巴並保持吐氣。
  • 用右手食指輕按壓住出氣按鈕模擬大量漏氣。
  • 將咬嘴咬一半，舌尖當擋水板，讓溢出的空氣可以排到水裡。
  • 頭側一邊，使氣泡不會影響你的視線。
  • 呼吸至少 30 秒。

# 跨步式入水

  • BCD 建立正浮力。
  • 咬著調節器。
  • 右手握著面鏡護著調節器。
  • 左手抓住錶組管子與 BCD 下擺邊緣。
  • 確認下水區沒有障礙物。
  • 下水後給岸上同伴一個 OK 的手勢，離開下水點，注視同伴下水。

# 中性浮力檢視

# 水中盤旋（使用低壓充氣閥）

  • 將 BCD 內空氣洩光，盤坐在水底。
  • 左手握著 BCD 充氣閥做點放式充氣。
  • 充完氣後開始呼吸（重複 3-4 次循環後）。
  • 直到吸氣時身體會開始上揚，吐氣時身體會慢慢下沉（蛙鞋不能碰觸地面）。
  • 不可憋氣，不可利用手腳晃動來維持平衡。
  • 至少 30 秒。

# 水中盤旋（使用口吹）

  • 將 BCD 內空氣洩光，盤坐在水底。
  • 吸一口氣後將二級頭拿離開嘴巴，左手將 BCD 充氣閥送入嘴巴，按住洩氣鈕並向內吹氣。
  • 吹完氣後將洩氣按鈕放開，並咬回調節器重新開始呼吸（重複 2-3 次循環後）。
  • 直到吸氣時身體會開始上揚，吐氣時身體會慢慢下沉（蛙鞋不能碰觸地面）。
  • 不可憋氣，不可利用手腳晃動來幫助平衡。
  • 至少 30 秒。

# 水底水肺裝備的脫卸與穿回

  • 確認 BCD 內部空氣洩光。
  • 先打開腰扣與黏扣帶。
  • 再打開左肩扣並將左手先穿出。
  • 右肩往前甩將裝備脫出。
  • 穿回前應將肩扣扣上。
  • 先穿右邊再穿左邊，並將腰扣與黏扣帶扣回。
  • 檢查管子有無糾纏。

# 水面水肺裝備的脫卸與穿回

  • 咬著呼吸管或調節器。
  • BCD 建立正浮力。
  • 先打開腰扣與黏扣帶。
  • 再打開左肩扣並將左手先穿出。
  • 右肩往前甩將裝備脫出（抓住不要放掉）。
  • 穿回前 BCD 先洩掉一些空氣，將肩扣扣上。
  • 先穿右邊再穿左邊，並將腰扣與黏扣帶扣回。
  • 檢查管子有無糾纏。

# 水底配重帶的脫卸與穿回

  • 將 BCD 內部空氣洩光。
  • 用右手打開快卸扣，將配重帶拉離開身體。右手抓住配重帶不要放掉。
  • 穿回時先用右手同時抓住帶尾與帶頭，保持快卸扣在左手邊的位置。
  • 檢查管子有無糾纏。

# 水面配重帶的脫卸與穿回

  • 咬著呼吸管或調節器。
  • 將 BCD 建立正浮力。
  • 用右手打開快卸扣，將配重帶拉離開身體。右手抓住配重帶不要放掉。
  • 穿回時先用右手同時抓住帶尾與帶頭，保持快卸扣在左手邊的位置。
  • 檢查管子有無糾纏。
