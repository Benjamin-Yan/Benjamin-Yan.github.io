# What are the differences between Intel i3, i5, i7, i9, and Xeon?

  • intel-Xeon 相比於 intel-Core (i3, i5, i7, i9) 的特色是處理器的核心數與執行緒較多、cache memory 較大且擴充性非常大。

    TypeSuited for價格功耗用途
    intel Core一般使用較便宜較低日常辦公、網頁瀏覽、遊戲、多任務處理
    intel Xeon工作站和伺服器所使用昂貴較大科學計算、工程模擬、資料中心

  • 其中，intel-Core 的數字愈大代表性能愈好，包括更多的核心數、更高的時脈速度、更大的快取容量，不過僅限於同一世代的比較。

    等級用途
    Core i3入門適合瀏覽網頁、文書處理
    Core i5中階適合輕度遊戲
    Core i7高階適合專業應用、遊戲、多媒體編輯和其他高性能任務
    Core i9頂級適合多核心應用、專業軟體和 3A 大作遊戲

# What are the differences between AMD Athlon, Ryzen 3, 5, 7, 9, Threadripper and Threadripper PRO?

共同點

  • 都是 AMD 的 CPU
  • 都有 multi-core & multi-thread 技術，以提高處理器的效能和多任務處理能力。

相異點

# What are the differences between DDR, DDR2, DDR3 and DDR4?

共同點

  • 它們是 RAM 的 DDR (Double Data Rate) 技術的不同世代。它們都遵循雙倍資料傳輸率的原則 (即在每個 clock cycle 中傳輸兩次資料，提高了資料傳輸效率)。

# What are the differences between SAS HD, SATA HD and SSD HD?

共同點

  • SAS & SATA 皆是電腦儲存資料的硬碟 (HD) 之介面規格

相異點

特性SASSATASSD
全名SAS（Serial Attached SCSI）SATA（Serial ATA）SSD（Solid State Drives）
Speed高速 (7200 RPM to 15000 RPM)較慢 (5400 RPM to 7200 RPM)非常高速，遠超 SAS 和 SATA (Read/Write Speed Up to 550 MB/s)
Storage600 GBMore than 2 TB120GB to 30.72TB
Power比 SATA 更耗能Power-efficient比 SATA 更耗能
Price較昂貴較便宜最貴

# What is Acorn RISC Machine architecture?

Acorn RISC Machine architecture (also known as ARM architecture), is a bunch of RISC architectures of CPU. It is suited for mobile device and more energy efficient.
