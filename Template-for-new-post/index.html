# Create new post

First open ..\hexo_blog_in_shoka_theme\source , and go to the proper place of your new post.
Then open cmd here, and type

如果沒有設定 layout，則會使用 _config.yml 中的 default_layout 來設定

Need to put title in "" if there are more than two words in it
(Because it will identify the first word as layout.)

# Required

Open the markdown file just created. In the metadata area, modify the following info.

  1. Use the default create date 即可
    date: 2023-04-19 13:33:45

  2. If only need one tag: tags: aaa

# Choice

  1. sticky: true if you want to 置頂一篇文章

  2. 可以指定網頁更新日期和時間 (若無，則會由系統自動帶入)，如下
    updated: 2023-04-20 23:31:00

  3. If more than two tags (can have space in a tag):

  4. Use cover image (file or web-image is ok)
    cover: https...\~.jpg or cover: assets/~.jpg

  5. Use categories to organize the related posts.

  6. Use mermaid: true to enable the flow chart. (P.S. see: 暫時解法)
