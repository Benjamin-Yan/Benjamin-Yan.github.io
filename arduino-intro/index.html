# Design Flow

  1. 決定你需要哪種類型的感測器。
  2. 學習如何使用感測器。
    • 了解感測器上每個引腳的功能。
  3. 設計你的專案電路 (可能需要一塊麵包板)
    • 確保電源（可能是 3.3V 或 5V）、接地（GND）和信號連接正確，以避免損壞元件。
  4. 學習感測器的 API 及如何使用它。
    • 閱讀感測器的庫文檔，了解可用的函數和方法以便與其互動。
    • 部分庫可以在 工具 → 管理庫 中設置。
  5. 編寫程式碼，使用 Serial.println() 進行除錯。
  6. 測試系統並完善設計。

# Examples

  • 超音波感測器
    • 使用聲波測量距離。
    • 任何需要知道距離的專案都可以從這個範例擴展。
  • 蜂鳴器
    • 主動蜂鳴器 (Active Buzzer)： 可用於警報系統等。
    • 被動蜂鳴器 (Passive Buzzer)： 可用於播放旋律。
  • RFID 系統
    • 你可以通過讀取或寫入卡片來構建 門禁控制系統 。
  • 其他感測器：使用感測器的 ID 搜尋其用途。
