# Arduino Project Proposal

# 1. 主題

智慧型手機使用提醒系統

# 2. 簡要說明:

這個專案旨在幫助用戶更有效地管理和改善手機的使用習慣，特別是針對長時間近距離觀看螢幕所可能引發的視覺疲勞、視力下降以及近視等健康問題。透過這個專案，用戶可以在靠螢幕太近時，獲得及時的提醒與建議，促進更健康的用眼習慣。

# 3. 目的

目的是幫助使用者保持視力健康，減少因長時間使用手機而引發的眼睛疲勞或近視等問題，從而提升生活品質。

# 4. 計畫用到的傳感器:

  • 超音波測距模組 (HC-SR04): 測量使用者與螢幕的距離，判斷是否會太接近螢幕。
  • 時間模組（DS1302): 紀錄長時間靠近螢幕的累積時間。
  • 光敏電阻：判斷環境光線，如果在黑暗中長時間使用手機，也會發出提醒並開啟 LED 小夜燈。
  • 蜂鳴器和液晶顯示器 (HLF8574T): 用來提醒使用者離手機螢幕太近了。
