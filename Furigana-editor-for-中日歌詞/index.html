# Instructions

  1. Go to the website where the bilingual lyrics exist.
  2. Copy the lyrics and separate the two language in each side of the textarea.
  3. Go to make furigana and change the left-side-lyrics (here is 日本語) into furigana html.
  4. 調整左側歌詞的分段，並記得 空兩行 / 段落間隔。
  5. Ensure the number of rows of the two language are the same!
  6. Click the Generate the output! button.
  7. Done! (Copy the necessary part that you want~)
