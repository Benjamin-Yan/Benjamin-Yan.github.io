# 上車與變速技巧

  • 上車時使用低速檔（前中配後中）。
    低速檔等引擎轉速夠再換檔，這樣可以保護雙腳並且保護變速器。

  • 停車時，應讓小對小讓變速器彈簧休息。
    這樣可以避免彈簧一直太緊繃，從而延長壽命。
    留在較大齒的檔位會對彈簧及變速線施加更多張力，彈簧撐久了會減弱其彈性，退檔速度與精準度也會下降。

  • 避免大對大檔位，因為這樣會對變速器鏈條造成損害。
    大盤應搭配 3 號檔位。

  • 變速線拉伸問題：長期使用會導致變速失準。

# 停車與腳踏車調整

  • 腳踏車高度調整：通常不會讓你停車時雙腳都能碰到地面，記得調整車座高度。

  • 檢查輪胎磨損：當圖檢查孔消失時，輪胎已經磨平。

# 騎行姿勢與技巧

  • 手臂與手掌姿勢：保持手臂放鬆，手肘微彎，手掌輕鬆放在變把上方。
    正常情況下，手不應該負擔太多重量。
    如果手部感到痠或麻，表示你手部承擔了過多體重。

  • 支撐體重：不要用手支撐體重，應該依靠脊椎與核心肌群支撐。

  • 視線：保持視線放遠，時刻注意前方車輛與號誌。

# 進階駕駛技巧

  • 逆操舵：逆操舵是單軌車種的進階技巧，使用場景通常有兩個：
  1. 快速轉向進入彎道
  2. 快速轉向閃避坑洞與障礙物。

# 其他注意事項

  • 大盤與大飛輪禁忌：不要使用大盤對大飛輪，或小盤對小飛輪，這樣會容易造成鍊條磨損。

  • 下坡煞車技巧：下坡時使用點放、點放的方式煞車，不要持續捏著煞車不放。
    寧可煞車稍微用力，再放開，再煞。

  • 轉彎信號：
    左轉時，舉起左手平舉；右轉時，舉起右手平舉，這樣可以讓後方車輛與其他車友知道你的轉彎方向。
